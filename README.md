# AspektCodeFix

A simple C# console application that searches through `.txt` files in a directory and appends custom text to them.

## Features
- Finds all `.txt` files recursively
- Appends text to every file
- Uses directory traversal with C#

## Technologies
- C#
- .NET 8
- Visual Studio

## How to Run

1. Open the solution in Visual Studio
2. Set the correct directory path in `Program.cs`
3. Run the project
